# Changelog

## [1.1.0] - 2026-07-11

- Rebuilt the practice and history UI as a responsive technical workspace.
- Added a deterministic, credential-free demo with an animated README walkthrough.
- Moved model execution from an in-process 8B model to hosted inference.
- Added strict provider-output validation and atomic quota updates.
- Hardened Clerk authentication, webhooks, CORS, error handling, and history bounds.
- Expanded frontend and backend tests and removed heavyweight local-model dependencies.

## [1.0.0] - 2026-06-23

- Polished public documentation and setup instructions.
- Removed generated SQLite user data from version control.
- Added security and code of conduct documentation.
- Added backend smoke tests and CI coverage for frontend and backend checks.
